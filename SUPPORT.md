# Support

If you need help using MorphCSS, there are a few ways to get support:

1. **Read the Documentation:** Our official documentation at [morphcss.dev](https://morphcss.dev) is the best place to start.
2. **GitHub Discussions:** Use our GitHub Discussions tab to ask questions, share ideas, and connect with other users.
3. **Open an Issue:** If you've found a bug or want to request a feature, please open an issue using the provided templates.

We kindly ask that you do not open GitHub Issues for general support questions. Use Discussions instead.
