# Security Policy

## Supported Versions

Currently, MorphCSS is in its Alpha phase. We only actively support and issue security patches for the `main` branch and the latest published NPM release. 

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in MorphCSS, please DO NOT open a public issue. 
Instead, please email the maintainers directly or use the GitHub Security Advisory private reporting tool.

We will respond to security reports within 48 hours and work with you to patch and disclose the issue responsibly.
